# 快速使用指南

## 基本命令

```bash
# 1. 最简单用法（自动检测团簇数量）
python analyze_ptsn_trajectory.py your_trajectory.xyz

# 2. 指定输出文件前缀
python analyze_ptsn_trajectory.py your_trajectory.xyz --output myrun_

# 3. 自定义载体原子数
python analyze_ptsn_trajectory.py your_trajectory.xyz --support 200

# 4. 完整参数
python analyze_ptsn_trajectory.py your_trajectory.xyz --support 240 --cluster 14 --output run1_
```

## 输出文件说明

| 文件名 | 用途 |
|--------|------|
| `surface_centered.xyz` | 载体居中 - 分析团簇表面迁移 |
| `cluster_centered.xyz` | 团簇居中 - 分析团簇内部重排 |
| `index_zsplit.ndx` | 原子组索引文件 |

## 常见问题

**Q: 团簇原子数怎么设置？**  
A: 默认自动计算（总原子数 - 载体数 = 团簇数），也可以用 `--cluster` 手动指定

**Q: 输出文件太大？**  
A: 可以减少帧数或压缩文件（gzip）

**Q: 出现警告"Cluster still fragmented"？**  
A: 如果最大距离接近 10 Å 是正常的，说明团簇本身是扩展结构

详细文档请查看 `README.md`
